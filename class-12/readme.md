# Class - 12

## Data Structure Coversion
- list(sequence): টাপল, সেট বা অন্য সিকোয়েন্স থেকে একটি নতুন লিস্ট তৈরি করে।
- tuple(sequence): লিস্ট, সেট বা অন্য সিকোয়েন্স থেকে একটি নতুন টাপল তৈরি করে।
- set(sequence): লিস্ট, টাপল বা অন্য সিকোয়েন্স থেকে একটি নতুন সেট তৈরি করে (ডুপ্লিকেটগুলো বাদ দিয়ে)।
- dict(iterable): কী-ভ্যালু পেয়ারের একটি সিকোয়েন্স থেকে ডিকশনারি তৈরি করে।
