# Class 22

## Public vs Protected vs Private
- Public
  - বাইরে থেকে এক্সেস করা যায়
- Protected _(single underscore)
  - বাইরে থেকে এক্সেস করা যায় তবে উচিত না
- Private __(double underscore)
  - বাইরে থেকে এক্সেস করা যায় না

## getter and setter
- Getter
  - read করা, get করা
- Setter
  - write করা, set করা
