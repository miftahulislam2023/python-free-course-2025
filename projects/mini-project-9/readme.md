# টপিক ৭০: মিনি প্রজেক্ট ৯: কন্টাক্ট বুক উইথ ফাইল স্টোরেজ (Mini Project 9: Contact Book with File Storage)

## রিকোয়ারমেন্টস:
- JSON ফাইল ব্যবহার করে ডেটা সেভ করতে হবে।
- প্রোগ্রাম চালু করার সময় ফাইল না থাকলে অটোমেটিক ফাইল তৈরি করে নিতে হবে (try-except ব্যবহার করে)।
- ফোন নম্বর ভ্যালিডেশন থাকতে হবে (সংখ্যা ছাড়া অন্য কিছু দিলে কাস্টম এরর raise করবে)।
- প্রোগ্রাম বন্ধ করে আবার চালু করলে পুরনো ডেটা থাকতে হবে।

### Contact Structure
- Name -> একই নামে একাধিক ব্যক্তি থাকতে পারবে
- Phone -> একই ফোন নাম্বার একাধিক ব্যক্তির থাকতে পারবে না
