# Class 17
## Exception handling
-  উদ্দেশ্য হল
   -  প্রোগ্রাম যেন ক্রাশ না করে
   -  অথবা ক্রাশ করলে যেন আমরা কী কারণে ক্রাশ করেছে তা জানতে পারি
-  TrEEFi
   -  Try -> যে কোডে error/exception আসতে পারে তা এই block এর ভিতরে লিখতে হবে
   -  Except -> error/exception আসলে কী করব তা এই block এর ভিতরে লিখতে হবে
   -  Else -> error/exception না আসলে কী করব তা এই block এর ভিতরে লিখতে হবে
   -  Finally -> error/exception আসুক বা না আসুক সবসময় রান হবে
