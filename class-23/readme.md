## Inheritance - উত্তরাধিকার
- Parent class/Base class/Super class
  - যে class থেকে attribute & method-গুলো নিব
- Child class/Derived class/Sub class
  - যে class attribute & method-গুলো নিবে
- Child class Parent class এর সবকিছু পেয়ে যাবে। শুধু তাই নয়, নিজে আর নতুন নতুন method বানাতে পারবে।
